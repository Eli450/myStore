## Installation
### To run the app with Node.js and postgres

1. Run `npm install`.

Wait for the build process to complete

1. Run `npm run dev` to start the api server



## Build the app
* Build manually
```
 $ npm start
```
Or
* Build with Docker Compose

```
$ docker-compose build
$ docker-compose up
```
